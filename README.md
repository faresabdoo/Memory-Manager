# Memory-Manager
محاكي إدارة الذاكرة (Memory Manager Simulator)

مشروع تعليمي تفاعلي يهدف إلى شرح وتوضيح مفاهيم إدارة الذاكرة في أنظمة التشغيل (Operating Systems) من خلال محاكاة مرئية لمختلف المخططات والأنظمة المستخدمة.

📝 وصف المشروع

يوفر هذا التطبيق واجهة مستخدم رسومية تتيح للطلاب والمتعلمين فهم كيفية تعامل نظام التشغيل مع الذاكرة الرئيسية (RAM). يغطي المشروع التطور التاريخي والتقني لأنظمة إدارة الذاكرة، بدءاً من الأنظمة البسيطة وصولاً إلى الأنظمة الأكثر تعقيداً.

🚀 الميزات الرئيسية

يتضمن المشروع محاكاة للمخططات التالية:

نظام المستخدم الواحد (Single-User Scheme): محاكاة لأبسط أشكال إدارة الذاكرة حيث يشغل مستخدم واحد الذاكرة بالكامل.

نظام الأقسام الثابتة (Fixed Partitions Scheme): توضيح كيفية تقسيم الذاكرة إلى أجزاء ثابتة الحجم وكيفية تخصيص المهام لها.

نظام الأقسام الديناميكية (Dynamic Partitions Scheme): عرض لعملية تخصيص الذاكرة بناءً على حجم الطلب الفعلي، مع شرح لمشاكل التجزئة الخارجية.

الأقسام الديناميكية القابلة للنقل (Relocatable Dynamic Partitions): محاكاة لعملية تجميع الذاكرة (Compaction) لحل مشكلة التجزئة.

مهام واختبارات (Tasks): قسم مخصص لاختبار فهم المستخدم من خلال مهام تطبيقية (من DSOS1 إلى DSOS5).

📂 هيكل الملفات

index.html: الصفحة الرئيسية للمشروع.

single-user-scheme.html: صفحة محاكاة نظام المستخدم الواحد.

fixed-partitions-scheme.html: صفحة محاكاة الأقسام الثابتة.

dynamic-partitions-scheme.html: صفحة محاكاة الأقسام الديناميكية.

relocatable-dynamic-partitions.html: صفحة محاكاة النقل وإعادة التخصيص.

admin.html: لوحة تحكم بسيطة (للإدارة).

Memory_manager_tasks.html: قائمة المهام التعليمية.

🛠 التقنيات المستخدمة

HTML5: لبناء هيكل الصفحات.

CSS3: لتنسيق الواجهات وجعلها متجاوبة.

JavaScript: للتحكم في منطق المحاكاة والعمليات الحسابية الخاصة بالذاكرة.

💻 كيفية التشغيل

قم بتحميل جميع ملفات المشروع في مجلد واحد.

افتح ملف index.html باستخدام أي متصفح ويب حديث (Chrome, Firefox, Edge).

تنقل بين الأقسام المختلفة من خلال القائمة الرئيسية.
