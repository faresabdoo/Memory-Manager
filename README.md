# 🧠 Memory Manager Simulator

مشروع تعليمي تفاعلي يهدف إلى شرح مفاهيم **إدارة الذاكرة (Memory Management)** في **أنظمة التشغيل (Operating Systems)** من خلال محاكاة مرئية لمختلف المخططات المستخدمة.

---

## 📖 About The Project

يوفر هذا التطبيق واجهة مستخدم رسومية (GUI) تساعد الطلاب على فهم كيفية تعامل نظام التشغيل مع **الذاكرة الرئيسية (RAM)**.

يعرض المشروع تطور تقنيات إدارة الذاكرة بدءًا من الأنظمة البسيطة وصولًا إلى الأنظمة الأكثر تقدمًا، مع تمثيل بصري يسهل الفهم.

---

## 🚀 Features

### 🧩 Memory Management Schemes

* **Single-User Scheme**

  * محاكاة أبسط نظام لإدارة الذاكرة حيث يستخدم مستخدم واحد كامل الذاكرة.

* **Fixed Partitions Scheme**

  * تقسيم الذاكرة إلى أقسام ثابتة وتخصيص العمليات داخلها.

* **Dynamic Partitions Scheme**

  * تخصيص الذاكرة حسب الحاجة مع توضيح مشكلة **External Fragmentation**.

* **Relocatable Dynamic Partitions**

  * دعم إعادة توزيع الذاكرة (Compaction) لحل مشكلة التجزئة.

---

### 🧪 Interactive Tasks

* مجموعة مهام تطبيقية من **DSOS1 إلى DSOS5**
* تساعد المستخدم على اختبار وفهم المفاهيم بشكل عملي
