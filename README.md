# 🧠 Memory Manager Simulator

🔗 **Live Demo:** https://memory-manger.netlify.app/
🔐 **Admin Panel:** https://memory-manger.netlify.app/admin.html

---

## 📖 About The Project

مشروع تعليمي تفاعلي يهدف إلى شرح مفاهيم **إدارة الذاكرة (Memory Management)** في **أنظمة التشغيل (Operating Systems)** من خلال محاكاة مرئية لأنظمة وتقنيات مختلفة.

يوفر التطبيق تجربة بصرية تساعد الطلاب على فهم كيفية إدارة **الذاكرة الرئيسية (RAM)** داخل النظام.

---

## 🚀 Features

### 🧩 Memory Management Schemes

* **Single-User Scheme**
* **Fixed Partitions Scheme**
* **Dynamic Partitions Scheme**
* **Relocatable Dynamic Partitions (Compaction)**

---

### 🧪 Interactive Tasks

* مهام تطبيقية (DSOS1 → DSOS5)
* اختبار الفهم بشكل عملي
